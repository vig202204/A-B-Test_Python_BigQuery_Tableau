# A-B-Test_Python_BigQuery_Tableau

## Опис Проєкту A/B Тестування та Аналіз Конверсії
Цей проєкт присвячений аналізу результатів A/B-тестування за допомогою статистичних методів в Python та створенню візуалізацій ключових конверсійних метрик. Метою є автоматизація розрахунку статистичної значущості для різних A/B-тестів та представлення результатів у зручному форматі, придатного для подальшої візуалізації, зокрема в Tableau.

## Ключові Особливості та Компоненти

### 1. Інтеграція з BigQuery
Проєкт підключається до Google BigQuery для отримання вихідних даних. Використовується складний SQL-запит для агрегації `сесій`, `подій`, `замовлень` та `нових облікових записів`, що забезпечує гнучкість у вибірці даних.

### 2. Обробка та Підготовка Даних
*   **Завантаження та Очищення:** Дані з BigQuery завантажуються в Pandas DataFrame. Стовпець `date` перетворюється на формат `datetime` для зручності аналізу часових рядів.
*   **Початковий Аналіз:** Виконуються базові перевірки даних (`head()`, `info()`, `describe()`, `isna().sum()`) для розуміння структури та якості набору даних.

### 3. Статистичний Аналіз A/B Тестування
*   **Визначення Метрик:** Аналізуються чотири ключові метрики конверсії: `add_payment_info`, `add_shipping_info`, `begin_checkout`, `new account`.
*   **Агрегація Даних:** Дані агрегуються за параметрами `test` та `test_group` для розрахунку загальної кількості сесій та подій по кожній метриці.
*   **Розрахунок Коефіцієнтів Конверсії:** Обчислюються коефіцієнти конверсії для кожної метрики шляхом ділення кількості подій на кількість сесій.
*   **Z-тест для Пропорцій:** Використовується `proportions_ztest` з бібліотеки `statsmodels` для розрахунку `p-значень`, порівнюючи пропорції конверсії між контрольною та тестовою групами для кожного `test_value`.
*   **Визначення Статистичної Значущості:** Результати позначаються як статистично значущі (`TRUE`) або незначущі (`FALSE`) на основі рівня `alpha = 0.05`.

### 4. Візуалізації в Python
Проєкт включає три основні візуалізації для представлення результатів:
*   **Групована стовпчаста діаграма:** Порівнює коефіцієнти конверсії контрольної та тестової груп для різних тестів та метрик, з індикаторами значущості.
*   **Стовпчаста діаграма відсоткової зміни:** Відображає відсоткову зміну конверсії між групами, з виділенням значущих результатів.
*   **Теплова карта P-значень:** Надає швидкий огляд статистичної значущості для всіх метрик та тестових значень.

### 5. Інтеграція з Tableau Public
*   **Експорт Даних:** Зведені результати експортуються у файл `df_ab_test.csv` та зберігаються на Google Drive, що дозволяє легко імпортувати їх у Tableau.
*   **Аналітичний Дашборд:** Розроблено інтерактивний дашборд у Tableau Public, який візуалізує ключові висновки та дозволяє глибше досліджувати дані.

## Результати та Висновки (Короткий Зміст)

*   **Тестове значення 1:** Усі чотири метрики конверсії (`add_payment_info`, `add_shipping_info`, `begin_checkout`, `new account`) показали статистично значущі покращення у тестовій групі порівняно з контрольною групою (p-значення в діапазоні від 0.000003 до 0.001646) при рівні `alpha 0.05`.
*   **Тестові значення 2, 3 та 4:** Жодна з метрик конверсії для цих тестових значень не показала статистично значущих відмінностей між контрольною та тестовою групами (усі p-значення були більшими за 0.05).

**Рекомендації:** Зосередитися на подальшому дослідженні змін, пов'язаних з `test_value = 1`, та аналізувати причини відсутності значущих результатів для інших тестів.

1.  **Налаштування Google Colab:** Відкрийте ноутбук у Google Colab.

5.  **Доступ до Дашборду Tableau:** Після виконання всіх кроків ви можете переглянути інтерактивний дашборд за посиланням, наданим у ноутбуці.
